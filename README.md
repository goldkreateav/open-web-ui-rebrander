# Open WebUI — модуль ребрендинга

## Структура каталогов

| Путь | Назначение |
|------|------------|
| `open-webui/` | Чистый клон [open-webui/open-webui](https://github.com/open-webui/open-webui) (ветка `main`) — референс и обновления с апстрима. |
| `REBRANDING_BASE.md` | Зафиксированная версия (коммит/тег), от которой ведётся ребрендинг. |

Дальнейшие изменения брендинга (логотипы, тексты, темы) можно вести в отдельной ветке внутри `open-webui/` или в копии каталога — по выбору на следующих этапах.
